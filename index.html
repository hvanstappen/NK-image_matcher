NK-DHM Beeldvergelijker

Deze tool helpt bij het visueel matchen van objecten uit de NK-collectie met afbeeldingen uit de DHM-set.

Navigatie

  • De resultaten zijn gegroepeerd per NK-nummer. Er is één pagina per object of objectset, bv. de pagina NK1393 geeft alle onderdelen van de set.
  • Zoek op NK-nummer (bv. NK1393 of NK1393-A) via de zoekbalk bovenaan
  • Rechts elke foto van het object worden de best gelijkende beelden uit de DHM collectie getoond.

Controle

  • Hou je muis boven ene beeld om het te vergroten
  • Klik op een NK-nummer om het object op de website van RCE te bekijken
  • Klik op een DHM nummer onder een foto om het record in de DHM database te bekijken.

Selectie

  • Klik op een DHM-beeld om een match vast te leggen. Je selectie wordt bijgehouden (zie teller recht boven)
  • Klik op 'Download selections ' om je selecties te downloaden als een CSV bestand.
  • Door op 'Clear all' te klikken worden alle selecties verwijderd..
